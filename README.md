# soccerGame_android
1.프로젝트 주제:  
  골키퍼(축구 공 막기)게임
	
2.개발환경 및 언어:  
  개발환경: Android Studio / 언어: Java

3.구현한 기능:
  1) 로그인 기능  
  
  2)  메뉴 선택 기능:  
    (a) 게임 시작하기  
    (b) 게임 알람 설정하기  
    (c) 로그아웃  
    
  3) 골키퍼 게임 진행을 위한 기능  
    (a) 캐릭터 선택 기능  
    (b) 게임 진행 기능  
    (c) 게임 초대 메시지: 카카오링크 v2 활용  
    
  4)게임 알람 기능:  
    상단바 알림 + 알람음
